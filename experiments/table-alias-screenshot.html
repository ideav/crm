issue #2967 — псевдоним таблицы (table alias) в js/integram-table.js

Обе таблицы построены реальным методом IntegramTable.parseObjectFormat() из собранного бандла js/integram-table.js. Различие — только наличие alias в метаданных.

ДО Без псевдонима

Метаданные: { val: "Клиент" } — заголовок и первая колонка показывают «сырое» имя термина.

ПОСЛЕ С псевдонимом

Метаданные: { val: "Клиент", alias: "Клиенты" } — заголовок и первая колонка показывают псевдоним, термин остаётся «Клиент».
