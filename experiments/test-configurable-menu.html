Тестирование настраиваемого меню (Issue #630)

Эта страница демонстрирует функциональность настраиваемого левого меню.

Функции:

  • Кнопка ☰ вверху слева - сворачивает/разворачивает меню
  • Кнопка ⚙ внизу слева - включает режим редактирования
  • В режиме редактирования:
    • Перетаскивайте пункты для изменения порядка
    • Перетащите на пункт для создания вложенности
    • Нажмите для настройки URL и иконки
    • Нажмите "Добавить пункт" для создания нового пункта
  • Фон меню: белый (Material Design)
  • Активный пункт: ненавязчивый серый цвет

API вызовы:

  • POST /_m_ord/{id}?JSON с order={номер} - изменение порядка
  • POST /_m_move/{id}?JSON&up={parent_id} - перемещение под родителя
  • Все запросы используют _xsrf токен
