Menu CRUD API Testing

This page documents the API endpoints for creating and deleting menu items as described in issue #632.

Create Menu Item

POST: /_m_new/151?JSON&up={parentId|roleId}

Parameters (form-urlencoded):

_xsrf
XSRF token from window.xsrf
t151
Menu item name (string)
t153
Menu item URL/href (string)
t391
Menu item icon (string - can be PrimeVue icon HTML or emoji)

Response:

{
    "obj": 123  // ID of the newly created menu item
}

JavaScript Implementation:

async function createMenuItem(name, href, icon, parentId) {
    const upParam = parentId || window.roleId;
    const url = '/_m_new/151?JSON&up=' + encodeURIComponent(upParam);

    const params = new URLSearchParams();
    params.append('_xsrf', window.xsrf || '');
    params.append('t151', name);
    params.append('t153', href);
    params.append('t391', icon);

    const response = await fetch(url, {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
        body: params.toString()
    });

    if (response.ok) {
        const data = await response.json();
        return data.obj; // Returns new menu item ID
    }
    throw new Error('Failed to create menu item');
}

Update Menu Item

POST: /_m_set/{menuItemId}?JSON

Parameters (form-urlencoded):

_xsrf
XSRF token from window.xsrf
t151
Menu item name (string)
t153
Menu item URL/href (string)
t391
Menu item icon (string - can be PrimeVue icon HTML or emoji)

Notes:

  • Uses the same field IDs as the create endpoint
  • All three fields (name, href, icon) should be provided

JavaScript Implementation:

async function updateMenuItem(menuId, name, href, icon) {
    const url = '/_m_set/' + encodeURIComponent(menuId) + '?JSON';

    const params = new URLSearchParams();
    params.append('_xsrf', window.xsrf || '');
    params.append('t151', name);
    params.append('t153', href);
    params.append('t391', icon);

    const response = await fetch(url, {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
        body: params.toString()
    });

    if (!response.ok) {
        throw new Error('Failed to update menu item');
    }
}

Delete Menu Item

POST: /_m_del/{menuItemId}?JSON

Parameters (form-urlencoded):

_xsrf
XSRF token from window.xsrf

Notes:

  • Backend automatically renumbers remaining items after deletion
  • All subordinate (child) menu items are also deleted automatically

JavaScript Implementation:

async function deleteMenuItem(menuId) {
    const url = '/_m_del/' + encodeURIComponent(menuId) + '?JSON';

    const params = new URLSearchParams();
    params.append('_xsrf', window.xsrf || '');

    const response = await fetch(url, {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
        body: params.toString()
    });

    if (!response.ok) {
        throw new Error('Failed to delete menu item');
    }
}

UI Integration

The menu edit mode is activated by clicking the settings button (gear icon) at the bottom of the sidebar.

In Edit Mode:

  • "Add item" buttons appear at the end of each menu level (including submenus)
  • Pencil (edit) button appears on each menu item - opens settings modal
  • Settings modal allows editing name, URL, and icon
  • Delete button appears in the settings modal (only when editing existing items)
  • Confirmation dialog shows when deleting, warning about child items if applicable
