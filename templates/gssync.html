Google Sheets Sync

Как пользоваться

1. credentials.json — сервисный аккаунт Google (скачивается в Google Cloud Console). Если путь не указан, используется служебная учетная запись Интеграма ideav-430@ideavpro.iam.gserviceaccount.com. Дайте ей доступ на чтение к нужной таблице.

2. Spreadsheet ID — из ссылки Google Sheets: docs.google.com/spreadsheets/d/ИДЕНТИФИКАТОР/edit

3. Sheets — JSON-массив листов. Каждый лист задает имя, список строк (метрик) и колонки в формате [from, to, label]. Пример:

[
    {
        "name": "(План-Факт) (2026)",
        "rows": ["Выручка *", "Поступления *"],
        "columns": [
            ["01.01.202*", "31.01.202*", "'ПЛАН'||'ФАКТ'"],
            ["01.02.202*", "'28.02.202*'||'29.02.202*'", "'ПЛАН'||'ФАКТ'"],
            ["01.03.202*", "31.03.202*", "'ПЛАН'||'ФАКТ'"],
            ["01.12.202*", "31.12.202*", "'ПЛАН'||'ФАКТ'"]
        ]
    }
]

4. Integram — выберите таблицу для загрузки (выпадающий список), укажите токен авторизации. autoParent — ID родительского типа. URL сервера и БД подставляются автоматически (текущий сайт/БД).

5. Логи (BKI-файлы) сохраняются в templates/logs/{_global_.z}/.

Конфигурации хранятся в templates/gss/{_global_.z}/.
