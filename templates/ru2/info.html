Добро пожаловать в Интеграм

Руководство по решению типовых задач. Нажмите на тему, чтобы раскрыть инструкцию.

Регистрация и начало работы

Как создать аккаунт и начать работу с Интеграмом:

  1. Перейдите на страницу регистрации и создайте аккаунт (через Google или Email).
  2. После регистрации будет создана ваша база данных.
  3. Войдите в личный кабинет — там вы можете управлять настройками аккаунта и паролем.
  4. На главной странице базы вы увидите навигатор данных — меню слева с разделами вашей базы.
Для восстановления пароля используйте функцию «Забыли пароль?» на странице входа.

Создание структуры данных (таблицы и поля)

Модель данных в Интеграме состоит из связанных между собой таблиц. Чтобы создать структуру:

  1. Откройте Редактор структуры (меню «Структура»).
  2. Создайте новую таблицу (термин) — задайте название и описание.
  3. Добавьте поля нужных типов: SHORT (короткий текст), CHARS (строка), MEMO (длинный текст), DATE (дата), NUMBER (число).
  4. Для выпадающих списков создайте справочные поля — они ссылаются на другую таблицу.
  5. Для вложенных данных (например, позиции заказа) создайте подчинённые таблицы.
Справочник — это отдельная таблица для выбора значений. Подчинённая таблица — это таблица, записи которой привязаны к конкретной записи родительской таблицы.

Работа с таблицами и записями

Основные операции с данными:

  • Добавление записи: откройте таблицу и нажмите кнопку добавления новой записи.
  • Редактирование: нажмите на запись, чтобы открыть форму редактирования.
  • Удаление: выберите запись и нажмите «Удалить». Система проверит, нет ли связанных данных.
  • Фильтрация: используйте маски и диапазоны для поиска нужных записей. Поддерживаются операторы >, <, >=, <= для чисел и дат.
  • Поиск в списках: в выпадающих списках работает поиск по введённым символам.
Используйте служебные слова: [TODAY] — текущая дата, [USER] — текущий пользователь.

Создание отчётов и запросов

Конструктор запросов позволяет строить отчёты без знания SQL:

  1. Перейдите в Конструктор запросов.
  2. Выберите таблицу-источник и нужные поля для отчёта.
  3. Добавьте условия фильтрации, сортировку и группировку.
  4. Используйте функции агрегации: COUNT (количество), SUM (сумма), AVG (среднее), MIN, MAX.
  5. Готовый отчёт можно экспортировать в Excel.

Расширенные возможности:

  • Интерактивные отчёты — с гиперссылками для перехода к записям.
  • Вложенные запросы — для сложной аналитики.
  • Параметры — передача параметров в отчёт через URL.
  • JSON-экспорт — для интеграции с внешними системами.

Конструктор форм и шаблоны

Интеграм использует систему пяти универсальных форм и HTML-шаблоны:

  1. Формы создаются в папке templates вашей базы.
  2. В шаблонах используйте блоки данных с фигурными скобками: {ИмяПоля} для вставки значений.
  3. Для доступа к родительским данным: {_parent_.ИмяПоля}.
  4. Для параметров запроса: {_request_.ИмяПоля}.
  5. Для глобальных констант: {_global_.ИмяКонстанты}.
Шаблоны можно редактировать локально и загружать через меню «Файлы».

Импорт и экспорт данных

Как загружать и выгружать данные:

  • Импорт из Excel: подготовьте таблицу с заголовками, соответствующими полям в Интеграме, и загрузите через меню импорта.
  • Формат BKI: специальный формат для обмена данными между базами Интеграма, поддерживает иерархическую структуру.
  • Нормализация данных: при загрузке плоских таблиц из Excel данные автоматически распределяются по связанным таблицам.
  • Экспорт: любой отчёт можно выгрузить в Excel или JSON.
При импорте данных из Excel рекомендуется сначала создать справочники, а затем загружать основные таблицы.

Настройка прав доступа

Система ролей позволяет гибко управлять доступом:

  1. Создайте роли для разных групп пользователей.
  2. Назначьте права на уровне таблиц — какие таблицы видны каждой роли.
  3. Настройте права на уровне полей — какие поля доступны для просмотра и редактирования.
  4. Используйте маски для ограничения по значениям — например, менеджер видит только свои записи.
  5. При необходимости настройте гостевой (анонимный) доступ.
Можно ограничить отдельные операции для ролей: экспорт, удаление, изменение определённых полей.

API и интеграция с внешними сервисами

Интеграм предоставляет API для взаимодействия с внешними системами:

  • Серверный API: вызовы с префиксом api/ для серверных операций.
  • JSON-запросы: получайте данные отчётов в формате JSON для использования в других приложениях.
  • Авторизация: используйте HTTP Basic Authentication с API-пользователем.
  • Внешние источники: подключайте данные из внешних HTTP-сервисов с помощью шаблонов запросов.
  • Power BI: подключайте отчёты Интеграма как источник данных в Power BI через JSON-экспорт.
Для работы с API создайте отдельного пользователя с паролем и назначьте ему роль с нужными правами.

Массовые операции с данными

Запросы модификации данных позволяют выполнять массовые операции:

  • Массовое обновление: изменение значений в нескольких записях одновременно с помощью запросов UPDATE.
  • Создание записей: автоматическое создание записей по условию с помощью CREATE.
  • Удаление: массовое удаление записей по фильтру с помощью DELETE.
  • Интеллектуальные отчёты: редактирование данных нескольких таблиц прямо в отчёте (как в Excel).
Перед массовыми операциями рекомендуется сначала проверить выборку через обычный отчёт.

Служебные функции и формулы

Встроенные функции для работы с данными:

  • abn_DATE2STR — преобразование даты в строку.
  • abn_NUM2STR — число прописью.
  • abn_ID — получение идентификатора текущей записи.
  • Стандартные MySQL-функции: CONCAT, SUBSTRING, IF, CASE и другие.
  • Кнопки (тип BUTTON): создание кнопок, которые выполняют действия с контекстом текущей записи. Поддерживают подстановку ID и VAL.
Полный список функций доступен в документации: help.integram.io.

Формы с оповещением в Телеграм

Как создать форму заявок с уведомлениями в Телеграм:

  1. Создайте таблицу для хранения заявок с нужными полями.
  2. Настройте HTML-форму с помощью конструктора форм.
  3. Подключите Телеграм-бота для отправки уведомлений о новых заявках.
  4. Настройте триггер — при создании новой записи бот отправит сообщение в указанный чат.
Это позволяет создать единую систему: форма на сайте, данные в реляционной базе и мгновенные уведомления в Телеграм.

Управление файлами

В Интеграме есть встроенная система управления файлами:

  • Загрузка файлов: используйте меню «Файлы» для загрузки файлов в базу.
  • Шаблоны: HTML-шаблоны хранятся в папке templates и редактируются через встроенный редактор или локально.
  • CSS и JS: стили и скрипты загружаются в соответствующие папки и подключаются к шаблонам.
  • Подключение файлов: используйте конструкцию <!-- File: имя_файла --> для включения одного шаблона в другой.
