Добро пожаловать в Интеграм

Это ваш экземпляр Интеграма — универсальной системы для работы с данными. Здесь вы можете создавать таблицы и запросы, строить формы, загружать данные из файлов, а также раздавать права другим пользователям.

Включить / отключить / сбросить режим подсказок

В каждом рабочем месте будет краткая информация по нему в виде 2–3 всплывающих окон. После первого показа информацию больше не показываем — если только режим подсказок не будет сброшен здесь.

Посмотреть какие таблицы есть в системе

Меню Таблицы. Даже в пустой системе уже есть базовые таблицы, которые нужны в любом приложении: пользователи, роли, запросы и связанные с этим справочники.

Перейти к таблицам
Загрузить данные из файла в таблицу

Меню Загрузка. Там можно загрузить файлы в существующую таблицу или создать новую таблицу прямо из файла в режиме помощника.

Перейти к загрузке
Раздать права пользователям в этой базе данных

Меню Пользователи. Здесь хранятся все пользователи системы. Изначально здесь только владелец этой базы данных, но сюда можно добавлять других пользователей и назначать им роли.

Перейти к пользователям
Посмотреть структуру данных

Меню Структура. Здесь можно быстро разобраться в структуре незнакомой базы данных — посмотреть все таблицы и их связи на одном экране, не листая отдельные таблицы и не изучая их список.

Перейти к структуре
Перейти на вкладку Формы и отчеты

Вкладка Формы и отчеты. Там хранятся популярные формы и отчеты для пользователей. Их можно сделать едиными для всех или привязать к пользовательским ролям, чтобы каждая роль видела свой набор запросов.
