Миграция конфигурации базы

Экспорт функционала текущей БД в JSON и развёртывание на другом инстансе. Выгружаются неслужебные таблицы с данными, шаблоны и скрипты, запросы, пользователи и пункты меню. Служебными считаются объекты с ID меньше порога. Доступно только роли admin.

2. Импорт / развёртывание

Термины переиспользуются, если свободны; занятые получают суффикс, чтобы не было конфликта имён и полей. Пароли не переносятся — импортированным пользователям нужно задать пароль.

3. Откат последнего импорта

Импорт ещё не выполнялся в этом браузере.
Удаляются все объекты и записи с ID ≥ ID первого созданного при импорте — записи, затем колонки, затем типы; несколько циклов с корректной обработкой ссылок (останов, если цикл ничего не удалил или циклов больше 100).
