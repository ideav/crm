Регистрация и начало работы

Интеграм позволяет быстро начать работу без сложной настройки:

  1. Регистрация доступна через Яндекс или Email — сразу на странице входа.
  2. После регистрации вам автоматически создаётся база данных — можно использовать как песочницу или сразу как полноценное приложение.
  3. В личном кабинете вы управляете настройками аккаунта и всеми своими базами.
  4. На главной странице базы доступен навигатор данных — меню слева со всеми разделами вашей базы.
Восстановление пароля доступно через функцию «Забыли пароль?» на странице входа.

Создание структуры данных (таблицы и поля)

Модель данных в Интеграме состоит из связанных между собой таблиц. Вы можете создать структуру любой сложности:

  1. Редактор структуры находится в меню «Структура» — там создаются и редактируются все таблицы.
  2. Создавайте новые таблицы (термины) — задавайте им человеческие названия и описания.
  3. Доступны поля разных типов: SHORT (короткий текст), CHARS (строка), MEMO (длинный текст), DATE (дата), NUMBER (число).
  4. Для выпадающих списков используйте справочные поля — они ссылаются на другую таблицу.
  5. Для вложенных данных (например, позиции заказа) доступны подчинённые таблицы.
Справочник — это отдельная таблица для выбора значений. Подчинённая таблица — это таблица, записи которой привязаны к конкретной записи родительской таблицы.

Работа с таблицами и записями

В Интеграме доступны все основные операции с данными:

  • Добавление записи: кнопка добавления новой записи доступна в каждой таблице.
  • Редактирование: щелчок по записи открывает форму редактирования со всеми полями.
  • Удаление: система автоматически проверяет наличие связанных данных перед удалением.
  • Фильтрация: поддерживаются маски и диапазоны для поиска, операторы >, <, >=, <= для чисел и дат.
  • Поиск в списках: в выпадающих списках работает поиск по введённым символам.
Доступны служебные слова: [TODAY] — текущая дата, [USER] — текущий пользователь и другие, которые помогут определить текущий контекст и тонко настроить поведение системы.

Создание отчётов и запросов

Конструктор запросов позволяет строить отчёты без знания SQL:

  1. Конструктор запросов доступен в главном меню — интуитивный визуальный интерфейс.
  2. Выбирайте любую таблицу-источник и нужные поля для отчёта.
  3. Настраивайте условия фильтрации, сортировку и группировку данных.
  4. Доступны функции агрегации: COUNT (количество), SUM (сумма), AVG (среднее), MIN, MAX.
  5. Готовые отчёты экспортируются в Excel одним нажатием.

Расширенные возможности:

  • Интерактивные отчёты — с гиперссылками для перехода к записям.
  • Вложенные запросы и рекурсия — для сложной аналитики и любых вычислений.
  • Параметры — передача параметров в отчёт через URL, служебные слова.
  • JSON-экспорт — для интеграции с внешними системами.

Конструктор форм и шаблоны

Интеграм использует систему пяти основных универсальных форм и произвольные HTML-шаблоны — вы полностью контролируете внешний вид:

  1. Формы хранятся в папке templates вашей базы и редактируются в любом текстовом редакторе.
  2. Для вставки значений полей используются блоки данных: {ИмяПоля}.
  3. Доступ к родительским данным: {_parent_.ИмяПоля}.
  4. Передача параметров запроса: {_request_.ИмяПоля}.
  5. Глобальные константы доступны через: {_global_.ИмяКонстанты}.
Шаблоны редактируются локально и загружаются через меню «Файлы» или из Git в рамках CI/CD процессов.

Импорт и экспорт данных

Интеграм поддерживает гибкий обмен данными с внешними источниками:

  • Импорт из Excel: загружайте таблицы с заголовками, соответствующими полям в Интеграме, через меню импорта.
  • Формат BKI: специальный формат для обмена данными между базами Интеграма с поддержкой иерархической структуры.
  • Нормализация данных: при загрузке плоских таблиц из Excel данные автоматически распределяются по связанным таблицам.
  • Экспорт: любой отчёт выгружается в Excel или JSON в один клик.
При импорте данных из Excel у вас есть все средства, чтобы правильно связать основные таблицы.

Настройка прав доступа

Система ролей обеспечивает гибкое управление доступом на всех уровнях:

  1. Создавайте роли для разных групп пользователей — с любыми наборами прав.
  2. Права настраиваются на уровне таблиц — каждая роль видит только нужные разделы.
  3. Права настраиваются на уровне полей — отдельно для просмотра и редактирования.
  4. Маски ограничивают доступ по значениям — например, менеджер видит только свои записи.
  5. Доступен гостевой (анонимный) режим для публичных форм и отчётов.
Для каждой роли ограничиваются отдельные операции: экспорт, удаление, изменение определённых полей.

API и интеграция с внешними сервисами

Интеграм предоставляет API для полноценной интеграции с внешними системами:

  • Серверный API: вызовы с префиксом api/ позволяют выполнять любые серверные операции.
  • JSON-запросы: данные любого отчёта доступны в формате JSON для использования в других приложениях.
  • Авторизация: поддерживается HTTP Basic Authentication с отдельным API-пользователем.
  • Внешние источники: данные из внешних HTTP-сервисов подключаются через шаблоны запросов.
  • Power BI: отчёты Интеграма подключаются как источник данных в Power BI через JSON-экспорт.
Для API-доступа создайте отдельного пользователя с паролем и назначьте ему роль с нужными правами.

Массовые операции с данными

Запросы модификации данных открывают широкие возможности для работы с большими объёмами записей:

  • Массовое обновление: значения в нескольких записях изменяются одновременно с помощью запросов UPDATE.
  • Создание записей: записи создаются автоматически по условию с помощью запросов CREATE.
  • Удаление: массовое удаление записей по любому фильтру с помощью запросов DELETE.
  • Интеллектуальные отчёты: данные нескольких таблиц редактируются прямо в отчёте (как в Excel).
Перед массовыми операциями выборку удобно проверить через обычный отчёт, который покажет, какие изменения будут внесены в данные.

Служебные функции и формулы

В Интеграме доступен широкий набор встроенных функций для работы с данными:

  • abn_DATE2STR — преобразует дату в строку нужного формата.
  • abn_NUM2STR — переводит число в текст прописью.
  • abn_ID — возвращает идентификатор текущей записи.
  • Все стандартные MySQL-функции: CONCAT, SUBSTRING, IF, CASE и другие.
  • Кнопки (тип BUTTON): кнопки выполняют действия в контексте текущей записи с подстановкой ID и VAL.
Полный список функций доступен в документации: help.integram.io.

Формы с оповещением в Телеграм

Интеграм позволяет создать единую систему: форма на сайте, данные в базе и мгновенные уведомления в Телеграм:

  1. Таблица для хранения заявок создаётся с любым набором полей.
  2. HTML-форма настраивается в конструкторе форм под нужный дизайн.
  3. Телеграм-бот подключается для отправки уведомлений о новых заявках.
  4. При создании новой записи бот автоматически отправляет сообщение в указанный чат.
Это no-code решение: форма на сайте, реляционная база данных и Телеграм-уведомления — без программирования.

Управление файлами

Интеграм имеет встроенную систему управления файлами для полного контроля над содержимым базы:

  • Загрузка файлов: меню «Файлы» открывает доступ к файловой системе базы прямо из браузера.
  • Шаблоны: HTML-шаблоны хранятся в папке templates и редактируются через встроенный редактор или локально.
  • CSS и JS: стили и скрипты загружаются в соответствующие папки и подключаются к шаблонам.
  • Включение файлов: конструкция <!-- File: имя_файла --> позволяет включать один шаблон в другой.
