## Что такое Интеграм?

Интеграм — это платформа для хранения, обработки и визуализации данных.

**Для кого подходит:**
- **Аналитику**, который устал от ограничений Excel
- **Менеджеру**, которому нужен удобный учёт без IT-отдела
- **Разработчику**, который хочет быстро собрать рабочий прототип

Если вам нужна база данных, к которой легко сделать запрос, отчёт или форму ввода — Интеграм создан именно для этого.

После входа в систему вы попадаете в личный кабинет:

## Данные: таблицы и связи

### Как устроена база данных в Интеграме

В Интеграме данные хранятся в **таблицах**, которые называются «типы». Каждый тип содержит **поля** (реквизиты) разных видов:

| Тип поля | Что хранит |
|----------|-----------|
| Текст (короткий) | Название, артикул, имя |
| Текст (длинный) | Описание, комментарий |
| Число | Цена, количество, сумма |
| Дата | Дата создания, срок |
| Флажок | Да/Нет, активен/неактивен |
| Файл | Вложенные документы, изображения |
| **Ссылка** | Связь с другой таблицей |

### Связи между таблицами

Ключевая возможность — поля могут **ссылаться на другие таблицы**. Так вы строите связанные структуры без SQL:

```
Заказы  →  Клиенты
   ↘
    Товары
```

Например:
- Таблица «Заказы» ссылается на «Клиентов»
- Таблица «Заказы» ссылается на «Товары»
- В результате вы видите полную картину по каждому заказу

### Как добавить данные

Данные загружаются тремя способами:
1. **Вручную через форму** — удобно для единичных записей
2. **Массово из CSV** — для импорта больших таблиц
3. **По API** — для автоматической интеграции с другими системами

Всё хранится в одном месте и остаётся взаимосвязанным.

> **Пример:** Создаём таблицу «Клиенты» с полями «Название компании», «ИНН», «Email», «Телефон». Затем создаём таблицу «Заказы» и добавляем в неё поле-ссылку на «Клиентов» — теперь каждый заказ знает, кому он принадлежит.

## Запросы без кода

### Конструктор запросов

Когда данные есть, их нужно искать и анализировать. В Интеграме для этого есть **конструктор запросов**:

1. Выбираете таблицу-источник
2. Добавляете фильтры (например: «статус = Активный»)
3. Указываете сортировку (например: «по дате, от новых к старым»)
4. Задаёте группировку (например: «по менеджеру»)

Никакого SQL, никаких скриптов.

### Сохранённые запросы

Запрос сохраняется как **«отчёт»**. Сохранённый отчёт можно:
- Открыть в любой момент одним кликом
- Поделиться ссылкой с коллегой
- Использовать как источник данных для рабочего места

> **Пример запроса:** «Все клиенты с типом "VIP", у которых срок договора истекает в течение 30 дней, сортировка по ответственному менеджеру».

## Отчёты, формы и опросы

На основе ваших данных и запросов в Интеграме можно создать три разных инструмента:

### 1. Отчёт

**Что это:** Таблица с выборкой данных, которую удобно просматривать и фильтровать прямо в браузере.

**Когда использовать:** Когда нужно регулярно смотреть на одну и ту же выборку данных — например, «активные клиенты» или «открытые задачи».

### 2. Форма ввода

**Что это:** Структурированная страница для добавления новых записей.

**Когда использовать:** Когда коллеги должны вносить данные, но не работают с полной таблицей. Форма показывает только нужные поля в удобном порядке.

> **Пример:** Менеджер по продажам заполняет форму «Новая заявка» — он видит только нужные ему поля, без технических столбцов системы.

### 3. Опрос

**Что это:** Форма, которую можно отправить **внешним пользователям** — клиентам, партнёрам.

**Когда использовать:** Когда нужно собрать данные от людей, у которых нет доступа к вашей системе. Ответы сразу попадают в вашу базу данных.

> **Пример:** Опрос удовлетворённости клиентов — клиент заполняет форму по ссылке, данные автоматически сохраняются в таблицу «Отзывы».

Все три инструмента собираются в конструкторах, **без кода**.

## Рабочие места с ИИ

### Что такое рабочее место

Рабочее место — это персонализированный дашборд прямо внутри Интеграма. Оно показывает данные именно так, как нужно конкретному сотруднику или команде.

### Как создать рабочее место

Процесс состоит из двух шагов:

**Шаг 1: Настройте отчёты**  
В конструкторе запросов создайте отчёты с нужными данными. Это источники данных для вашего рабочего места.

**Шаг 2: Создайте шаблон**  
В шаблонизаторе опишите, как должно выглядеть рабочее место: какие данные показывать, как их отображать. **ИИ помогает сверстать нужный HTML-шаблон** — достаточно объяснить задачу словами.

Шаблон подключается к отчётам и **автоматически берёт актуальные данные**.

### Где находится рабочее место

Готовое рабочее место добавляется в меню «Файлы». Любой сотрудник видит свой персонализированный дашборд прямо в Интеграме — без внешних сервисов и дополнительных инструментов.

> **Пример:** Руководитель отдела продаж видит рабочее место с ключевыми метриками: количество сделок за месяц, воронка по менеджерам, топ-клиенты. Менеджер видит своё место: его задачи на сегодня, его клиенты, его статистика.

## API: обмен данными в обе стороны

Интеграм открыт через **JSON API**. Это значит, что система легко встраивается в существующие процессы компании.

### Что можно делать через API

| Действие | Описание |
|----------|----------|
| Читать данные | Выгружать данные из отчётов в любой внешний сервис |
| Писать данные | Загружать данные из других систем в таблицы Интеграма |
| Управлять структурой | Создавать и изменять таблицы и поля программно |

### Пример запроса

```http
GET /api/report/12345
Authorization: Bearer <token>
```

Ответ приходит в формате JSON — стандарте, который поддерживают все современные языки программирования и инструменты.

> **Практический сценарий:** Интернет-магазин автоматически отправляет новые заказы в Интеграм через API. Менеджеры видят заказы в удобной таблице, работают с ними, а результаты через API возвращаются обратно в магазин.

## Быстрый старт

Интеграм — это ваша база данных, конструктор запросов, генератор форм и движок для рабочих мест в одном инструменте.

### Попробуйте прямо сейчас — 3 шага за 5 минут

**Шаг 1: Создайте таблицу**
1. Откройте раздел «Базы данных»
2. Нажмите «Создать тип»
3. Добавьте несколько полей: название, дата, статус

**Шаг 2: Добавьте данные**
1. Откройте созданную таблицу
2. Нажмите «Добавить запись» и заполните поля
3. Добавьте 3–5 записей для начала

**Шаг 3: Сделайте запрос**
1. Откройте конструктор запросов
2. Выберите вашу таблицу
3. Добавьте фильтр или сортировку
4. Сохраните как отчёт

Это займёт несколько минут — и вы сразу увидите, как работает система.
