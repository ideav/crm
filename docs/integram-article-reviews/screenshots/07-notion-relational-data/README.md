| Файл | Что на экране | Где используется |

## Чек-лист перед добавлением

- На скриншотах нет реальных персональных данных клиентов и сотрудников (используется демонстрационный набор).
- Имена ролей, полей и таблиц на демо-стенде осмысленные («Менеджер», «Сделки», «Клиенты»), а не «Test 1», «Test 2».
- Размер изображений не превышает разумного для документации — желательно PNG до 500 КБ.
- Текст на скриншотах читается на ширине статьи (для DPI ≥ 1.5 пересохранять при необходимости).
