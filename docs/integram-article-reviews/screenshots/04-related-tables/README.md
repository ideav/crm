## Требования к скриншотам

- Формат: PNG, ширина 1280–1920 px.
- Содержимое — обезличенные демо-данные, без реальных контактов клиентов.
- Подсветка ключевого элемента (стрелка, рамка) — допустимо, но не обязательно.
- Имя файла — английскими буквами по схеме `NN-описание.png` (как в таблице выше).
