# Скриншоты для статьи 09

Папка для визуальных материалов к статье [`../../09-custom-development-prototype.md`](../../09-custom-development-prototype.md).

## Сценарий

Демонстрационный процесс — приём заявок от партнёров. Все скриншоты делаются на одном и том же демо-стенде, чтобы переходы между экранами выглядели как один процесс.

## Требования к скриншотам

- Разрешение: не меньше 1280×800, без артефактов масштабирования.
- Демо-данные: партнёры — реальные названия категорий («Логистика», «Сервис», «Доставка»), без персональных данных.
- Скриншоты делаются последовательно на одной сессии, чтобы видна была единая стилистика интерфейса.
- При замене скриншота не менять имя файла — ссылки в статье останутся валидными.
