## Соответствие раскадровке видео

- Хук 0:00–0:10 — `01-excel-row-limit-error.png`.
- Демонстрация 0:10–0:35 — `02-integram-import-excel.png`, `03-integram-related-tables.png`, `04-integram-table-large-dataset.png`.
- Сравнение 0:35–0:55 — `04-integram-table-large-dataset.png` и `05-integram-export-excel.png` рядом (или быстрая нарезка).
- Следующий шаг 0:55–1:00 — текстовая карточка со ссылкой на инструкцию по импорту, отдельный скриншот не нужен.

## Технические требования

- Формат: PNG, без потери качества.
- Разрешение: не меньше 1280×720, желательно 1920×1080 для итогового видео.
- В кадре не должно быть персональных данных реальных клиентов — использовать демо-данные.
- Подсвечивать ключевой элемент (рамка/обводка) только если это явно помогает восприятию; иначе оставлять чистый скриншот.
