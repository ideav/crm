## Требования к скриншотам

- Формат: PNG, ширина 1280–1920 px.
- Чувствительные данные обезличить (имена сотрудников, телефоны, адреса).
- Имя файла — точно как в таблице выше; статья ссылается на эти имена.
- При замене шаблона на финальный — оставить тот же сценарий (учёт заявок на сервисное обслуживание), чтобы скриншоты были согласованы между собой.
