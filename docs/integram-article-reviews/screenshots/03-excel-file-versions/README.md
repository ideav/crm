# Скриншоты к статье 03 «Единая версия данных»

Папка содержит визуальные материалы к [`../../03-excel-file-versions.md`](../../03-excel-file-versions.md) и к 1-минутному видео-обзору.

## Статус

Файлы в формате SVG — это **раскадровка-макеты**: каждый макет показывает, что должно быть на экране, в виде стилизованного UI. Они нужны для согласования сценария и видеосъёмки до того, как появятся реальные скриншоты Интеграма с подготовленным демо-набором данных.

Перед публикацией статьи или съёмкой 1-минутного видео макеты заменяются на реальные PNG-скриншоты Интеграма с теми же именами файлов (`01-…`, `02-…` и т.д.) и тем же смыслом каждой сцены.

## Как заменять макеты на реальные скриншоты

1. Подготовить демо-базу с теми же сущностями: контрагенты, договоры, платежи, роли «менеджер», «руководитель», «финансист».
2. Сделать PNG-скриншоты в порядке макетов и с теми же именами, заменив расширение на `.png`.
3. Удалить соответствующий SVG-макет либо оставить рядом (полезно для повторных съёмок).
4. Проверить, что в статье и в раскадровке видео ссылки и описание сцен остались согласованными.
