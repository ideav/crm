## Требования к скриншотам

- Формат: PNG, ширина 1280–1920 px.
- Чувствительные данные обезличить: реальные имена клиентов, суммы и номера договоров заменить на демо-значения.
- Имя файла — точно как в таблице выше; статья и раскадровка видео ссылаются на эти имена.
- Все скриншоты делать на одном демо-наборе данных (реестр сделок торговой компании), чтобы между кадрами легко прослеживалась одна и та же запись.
- Токены API в кадрах `02-api-endpoint.png` и `04-powerbi-connection.png` обязательно маскировать (`••••` или сокращённый префикс).
