# Скриншоты к статье 05: Права доступа

Файлы в этой папке используются в статье `docs/integram-article-reviews/05-access-rights.md` для иллюстрации ключевых сценариев.

| Файл | Что иллюстрирует | Где в статье |
| --- | --- | --- |
| `01-google-sheets-single-level.svg` | Диалог «Настройки доступа» в Google Sheets: все пользователи получают одинаковый уровень «Редактор». | Шапка статьи. |
| `02-integram-role-mask.svg` | Экран роли «Менеджер» в Интеграме: WRITE/READ/BARRED по таблицам и полям, маска `Ответственный = [CURRENT_USER]`. | Раздел «Что делает Интеграм иначе». |
| `03-manager-workspace.svg` | Рабочее место менеджера: только свои сделки, колонка «Сумма» помечена как скрытая (BARRED). | Раздел «Маски по значениям». |
| `04-guest-public-form.svg` | Публичная форма заявки, открытая без логина под встроенной ролью `guest`. | Раздел «Гостевой доступ». |

Все четыре файла — SVG-мокапы, собранные на основе реальной модели доступа Интеграма (см. `index.php`, функции `getGrants`, `Check_Grant`, `Construct_WHERE`). Перед публикацией статьи в блоге их рекомендуется заменить на скриншоты живой системы с тестовыми данными, сохранив те же сцены и подписи.
