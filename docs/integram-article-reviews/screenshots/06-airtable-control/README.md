# Скриншоты к статье 06 — «Собственный контур вместо SaaS-зависимости»

Эта папка предназначена для скриншотов и кадров из 1-минутного видео-обзора по теме статьи [`06-airtable-control.md`](../../06-airtable-control.md).

Скриншоты в репозиторий не добавляются автоматически: они снимаются на реальном развёрнутом Интеграме перед публикацией статьи и видео.

## Что нужно снять

Минимум — 3 кадра под раскадровку видео и 1 кадр с тарифами Airtable для хука.

| Файл | Что на экране | Где взять |

## Правила оформления

- Формат: `.png` (по умолчанию) или `.webp` для крупных кадров.
- Перед публикацией затереть/замазать любые персональные данные клиентов и сотрудников (имена, email, телефоны, суммы).
- Имена файлов — латиницей, через дефис, с порядковым префиксом (`NN-краткое-описание.png`).
- Числа на скриншоте тарифов Airtable должны совпадать с числами в тексте статьи. Если Airtable изменил лимиты — обновить и скриншот, и текст.
