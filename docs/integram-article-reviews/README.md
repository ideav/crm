# Статьи-обзоры Интеграма

Эта папка предназначена для будущих статей-обзоров по плану серии из [docs/INTEGRAM_ARTICLE_SERIES_PLAN.md](../INTEGRAM_ARTICLE_SERIES_PLAN.md), сопутствующих скриншотов и материалов для сокращенного формата. Для каждой темы создан отдельный GitHub issue: в нем нужно подготовить саму статью, экранные примеры и 1-минутный видео-обзор.

## Структура

- `NN-slug.md` - черновик или готовый текст статьи.
- `screenshots/NN-slug/` - скриншоты и другие визуальные материалы для статьи.
- GitHub issue - отдельная задача на подготовку текста, скриншотов и 1-минутного видео-обзора.

## Минимальный состав каждой задачи

- Статья раскрывает один конкретный сценарий из плана серии.
- Скриншоты показывают структуру, таблицу, форму, отчет, дашборд, права доступа или другой экранный пример из темы.
- 1-минутный видео-обзор содержит короткий хук, 2-3 демонстрационных экрана и итоговый вывод.
- Численные сравнения и лимиты перепроверены перед публикацией.
